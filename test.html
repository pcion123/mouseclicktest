滑鼠檢測工具 - 單機測試指南

1. 基本載入測試

✅ 打開 主應用程式

✅ 檢查所有樣式是否正確載入

✅ 確認所有按鈕和控制項可見

2. 功能測試

A. 基本點擊測試

  1. 點擊「開始測試」按鈕
  2. 在藍色測試區域內點擊 10 次
  3. 觀察統計面板數據更新
  4. 檢查圖表是否繪製點擊軌跡
  5. 點擊「停止測試」

B. 異常檢測測試

  1. 重複點擊測試：快速雙擊多次，應看到紅色波紋
  2. 連擊測試：500ms 內點擊 3 次以上
  3. 間隔測試：觀察點擊間隔統計

C. 模式切換測試

  1. 切換到「節奏測試」模式
  2. 切換到「耐久測試」模式
  3. 觀察模式描述變化

3. 設定功能測試

  1. 點擊「設定」按鈕打開設定對話框
  2. 修改檢測參數（如重複點擊窗口）
  3. 保存設定
  4. 測試新設定是否生效
  5. 重新整理頁面檢查設定是否保留

4. 瀏覽器相容性測試

在以下瀏覽器中測試：

  • ✅ Chrome
  • ✅ Firefox
  • ✅ Edge
  • ✅ Safari (如果可用)

5. 性能測試

  1. 進行 100+ 次點擊測試
  2. 觀察記憶體使用情況
  3. 檢查響應速度是否保持流暢
  4. 測試長時間運行穩定性

6. 數據功能測試

  1. 完成一次測試會話
  2. 檢查統計報告是否正確
  3. 驗證健康度評估是否合理
  4. 測試數據清除功能
